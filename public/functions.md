# MemFlow AI Core Functionality

This project provides the following user-facing functionality:

## 1. Learn C Programming

- Browse structured C lessons from basics to advanced topics.
- Open topic pages for concepts such as variables, loops, functions, pointers, memory, file I/O, and more.
- Search topics from the sidebar and jump into suggested lessons from the home screen.

## 2. Write And Run C Code In The Browser

- Edit C programs in the built-in code editor.
- Run code directly from the app and view the output in the terminal panel.
- Stop a run in progress when needed.
- Provide runtime input for programs that use `scanf`.

## 3. Visualize Execution Step By Step

- Capture execution as a timeline of steps.
- Move forward and backward through execution with step controls.
- Highlight the active source line while stepping through the program.

## 4. Understand Memory Behavior

- View variable values as they change over time.
- Inspect memory cards, memory maps, and stack/heap style views.
- See pointer relationships visually, including links between referenced values.
- Open a dedicated memory overlay for deeper inspection.

## 5. Get AI Assistance

- Ask general C programming questions in the chat area.
- Ask AI to explain selected lines from the editor.
- Ask AI to explain the current execution step and memory changes.
- Continue those explanations inside the shared chat workspace.

## 6. Track Learning Progress

- Sign in and keep a per-user profile.
- Store theme and preferred language choices.
- Track completed topics and progress percentage.
- Show recommended topics based on progress.

## 7. Use The App In Multiple Languages

- Switch the UI between English, Hindi, Bengali, Telugu, Tamil, and Marathi.
- Load language-specific UI text packs.
- Support Hindi lesson content in addition to the default lesson set.

## 8. Save Or Export What The User Is Viewing

- Download the current C code as a `.c` file.
- Download the currently opened lesson/topic as a standalone HTML file.
- Preserve chat sessions for the signed-in user.
